# Python Data Analytics Projects

This repository contains data analytics projects where I analyze real-world datasets to extract meaningful insights using Python.

## Projects

### 1. Exploratory Data Analysis (EDA)
- Performed data cleaning, preprocessing, and exploration
- Identified trends, outliers, and patterns using visualizations

### 2. Skills Trend Analysis
- Analyzed in-demand technical skills across job roles
- Visualized skill popularity and trends using charts

### 3. Salary Analysis
- Studied salary distributions across roles and locations
- Compared salaries based on experience and skills
- Derived insights to understand market trends

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

- Real-world data cleaning and handling missing values
